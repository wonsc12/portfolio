데이터베이스  데이터 추가/읽기/수정/삭제 명령어 복습 및 진도

insertOne() 하나의 객체 데이터 쓰기
find() 다수의 데이터 읽기   findOne() 하나의 객체 데이터 읽기
updateOne() 하나의 객체 데이터 수정
deleteOne() 하나의 객체 데이터 삭제

해당 함수들은 뒤에 콜백 함수 function(err,result) 추가
데이터 처리 후 뒤에 실행되는 콜백함수를 추가한다는 뜻
err는 에러메세지 출력시 필요한 매개변수 
result는 데이터(객체형태) 값 자체를 뜻함
